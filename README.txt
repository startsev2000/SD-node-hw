Небольшая инструкция

Для начала нужно выполнить команду npm install ws (или можно просто npm install, т.к. есть файл package.json) - она нужна для установки нодовского пакета, в котором содержатся методы для работы с вебсокетами.

Далее необходимо выполнить команду node server.js - она запустит сервер.
Для проверки работоспособности можно выполнить команду node client.js - эта программа отправит базовую строку, она должна будет отобразиться на сервере.

Запускать программы нужно в разных терминалах :)

Удачи и спасибо!
